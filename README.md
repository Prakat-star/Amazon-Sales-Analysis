### Amazon Product Data Analysis using SQL and Python

## Tools: 
Python, Pandas, SQLite, Seaborn, Matplotlib, and dataset from kaggle
## Description:
    Analyzed Amazon product data to understand pricing, discounts, ratings, and reviews. Combined SQL queries with Python for data cleaning, aggregation, and visualization.

## Objectives:
    -Identify most expensive and cheapest products
    -Explore discount patterns across categories
    -Study rating trends and top-reviewed products
    -Visualize correlations between price, discount, and ratings

## Highlights:
    -Top 10 most expensive and cheapest products
    -Top 10 products with most reviews
    -Average discount per category
    -Correlation heatmap of price, discount, rating, and reviews
